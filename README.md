# alfa-digital-test-task

## Run Locally

Clone the project

```bash
  git clone https://github.com/MOONcitizenX/vite-react-ts.git
```

Go to the project directory

```bash
  cd alfa-digital-test-task
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```

## Build

To build this project run

```bash
  npm run build
```
